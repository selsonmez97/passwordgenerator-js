Need Help?

  1. Click the "Generate passwords" button.
  2. Choose which one you'd like to keep by highlighting the password inside of the input box.
  3. Then right click and select "copy".
  4. If you don't like any of them feel, free to re-generate new passwords by clicking the "Generate passwords" button again.
  5. You can also edit any part of the password by clicking inside of the password text or input box.
